## 一些使用小技巧

用了几个月，我摸索出一些经验：

**文本准备**：在转换之前，最好先调整一下文本格式。去掉那些不需要读出来的符号、链接，段落之间留点空行，这样生成的音频停顿会更自然。

**试听对比**：同一段文本，可以用不同的声音角色生成几个版本，对比一下效果。有时候换个声音，整个感觉就不一样了。

**分段处理**：如果内容特别长，建议分成几段分别处理。一是可以针对不同部分调整参数，二是万一哪段不满意，只需要重新生成那一段，不用整个重来。

**保存设置**：当你找到满意的声音和参数组合后，可以记下来，👉[下次直接使用](https://ttsopenai.com)，省得每次都重新调整。
